## ✨ 주요 기능

- 한글 자모 6개로 단어 추측
- 초성/중성/종성 직접 분해 및 조합 로직 구현
- 성공 시, 결과 복사하여 친구와 공유
- 실패 시 "텅 빈 밥그릇" 등장 🍚
- 자체 제작한 일러스트 기반의 귀여운 스토리 & 튜토리얼
